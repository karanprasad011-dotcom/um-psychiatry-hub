⚙️ How to manage all your content

1

Create a new Google Sheet

Go to sheets.google.com and create a new spreadsheet. Name it "UM Psychiatry Research Hub". Rename the first tab (bottom left) to exactly: Projects

2

Add the column headers in Row 1

In the very first row, add these column names exactly as shown (one per cell, left to right):

id  title  pi  contact  description  specialty  level  commitment  status  tag  timeline  skills

3

Add your first project in Row 2

Fill in the columns for your first project. See the column guide on the right for what to put in each one. For id, just type 1. For status, type either open or closed.

4

Publish the sheet to the web

Click File → Share → Publish to web. Under "Link", select the Projects tab and choose Comma-separated values (.csv). Click Publish and copy the URL it gives you.

5

Paste the URL above and click Connect

Paste the CSV URL into the connection box at the top of this page and click Connect. Your projects will appear on the live site within a few minutes of any edits.

6

Adding or editing projects going forward

Just open your Google Sheet and add a new row for each project, or edit existing rows. Changes appear on the live site automatically — no redeployment needed. Each new project needs a unique number in the id column (1, 2, 3...).
